### Summary Statistics Columns:
- SNP: rsID
- CHR: Chromosome
- BP: Position in build 37
- A1: Effect allele
- A2: Non-effect allele
- EAF: Effect allele frequency
- MAF: Minor allele frequency
- Beta: Effect size of the EAF allele
- Se: Standard error
- P: P-value
- Z: Z-score
- N: Total sample size
- Direction: Direction of effect
- HetI2: Heterogeneity I^2
- HetP: Heterogeneity P-value
